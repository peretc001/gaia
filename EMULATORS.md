# Firebase Local Emulator Suite

## Запуск эмуляторов

Для запуска Firebase Local Emulator Suite выполните команду:

```bash
firebase emulators:start
```

Это запустит:
- **Auth Emulator** на порту `9099`
- **Emulator UI** на порту `4000` (веб-интерфейс для управления эмуляторами)

## Использование

1. Запустите эмуляторы: `firebase emulators:start`
2. Запустите Flutter приложение в debug режиме
3. Приложение автоматически подключится к эмуляторам

## Веб-интерфейс эмуляторов

Откройте в браузере: http://localhost:4000

Здесь вы можете:
- Просматривать пользователей, созданных в Auth Emulator
- Управлять данными эмуляторов
- Просматривать логи

## Создание тестовых пользователей

В эмуляторе вы можете создавать пользователей через:
1. Веб-интерфейс (http://localhost:4000)
2. Код приложения (регистрация через форму)

## Остановка эмуляторов

Нажмите `Ctrl+C` в терминале, где запущены эмуляторы.

## Важно

- Эмуляторы работают только в **debug режиме**
- Все данные в эмуляторах хранятся локально и удаляются при остановке
- Для production используйте реальный Firebase проект
